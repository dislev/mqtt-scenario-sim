# Security Policy

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Use GitHub's private vulnerability reporting instead:
**[Report a vulnerability](https://github.com/dislev/mqtt-scenario-sim/security/advisories/new)**

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 7 days. This project is maintained on a best-effort basis — critical issues will be prioritised.
